---
"subject": subject code as defined in source files subject_test.txt or subject_train.txt 

"activity": activity name as defined in source activity_labels.txt - derived from activity codes contained in source files y_test.txt or y_train.txt

"variable": one of the standard deviation ("std") or mean measures contained within source files x_test.txt or x_train.txt

"avg": average of all values present in the source data for the specified (subject, activity, variable) combination
---

For more information refer to the source data:
https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip
